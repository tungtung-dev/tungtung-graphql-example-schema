## Examples blog post with graphQL
Small examples, best pratices for Node.js & graphQL

## Installation

```
yarn install # or
npm install
```

## Run Dev

```
npm run dev 
# go to localhost:8080/graphql
```

## Build produciton

```
npm run build
npm run start
# go to localhost:8080/graphql
```

## Content seeder
If you want auto generator fake data, please run URL
```
http://localhost:8080/seeder
```
